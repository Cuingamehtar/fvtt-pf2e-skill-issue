# PF2e/SF2e Skill Issue

This module simplifies Pathfinder 2e and Starfinder 2e manual skill increases by keeping track at which levels they were made. The skill manager then issues the appropriate proficiency level instead if the one in source data to be further processed by rule elements.
Supports both core skills and lores, which are tracked by their name.

## Workflow

1. Press the "Skill manager" button on the character's proficiencies tab to open the skill manager
<img width="511" height="221" alt="si-1" src="https://github.com/user-attachments/assets/82d75d94-d811-4c48-990c-d707a8bc1afb" />

2. Configure the skill upgrades the character gets. User "override" column to do manual adjustment if necessary, and keep notes in a text field at the bottom of the window.
